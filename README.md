# wechat-cloud-demo
个人微信小程序，包括工具实现、功能实现、个人简介和云开发示例。
博客地址：[我的微信小程序](https://www.yuque.com/luowenshuai/learn/yr2s3g "我的微信小程序")

#### 工具
- 日历管理（基于 Data 对象）

#### 功能
- 添加水印（基于 svg）
- 列表滚动加载

#### 云开发
- 列表
- 新增
- 查看
- 编辑
- 删除

# wechat-cloud-demo
个人微信小程序，页面包括工具实现、功能实现、个人简介和云开发示例，使用自定义tabBar、分包功能。
博客地址：[我的微信小程序](https://www.yuque.com/luowenshuai/learn/yr2s3g "我的微信小程序")

#### 工具
- 日历管理（基于 Data 对象）

#### 功能
- 自定义水印（基于 svg）
- 列表滚动加载

#### 云开发列表
- 列表
- 新增
- 查看
- 编辑
- 删除

#### 视频播放

#### 个人简介
- 使用云开发存储数据和文件
- 动态展示内容
- 使用分包加载
- 支持详细版和简洁版切换
